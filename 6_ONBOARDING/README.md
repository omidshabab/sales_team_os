Title: Onboarding Checklist
Owner role: RevOps
Last updated: 2025-12-25
Purpose: Get tools, calendars, and CRM configured in one day.
How to use (1-2 lines): Perform steps sequentially; mark complete; proceed to SOPs.

1. Accounts — **RevOps** (30m): CRM, email, calendar, Zoom.
2. Calendar — **RevOps** (20m): public booking link; connect to CRM.
3. CRM fields — **RevOps** (45m): import `crm_mappings.yml`.
4. Templates — **RevOps** (30m): load `/8_TEMPLATES/` into sequences.
5. Dashboards — **RevOps** (30m): add queries from profile `/dashboards/`.
6. KPI targets — **Head of Sales** (30m): set weekly targets.
7. Role training — **Manager** (2h): run `learning_path/` micro-lessons.
