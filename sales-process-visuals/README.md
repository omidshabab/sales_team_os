Title: Sales Process Visuals
Owner role: RevOps Lead
Last updated: 2025-12-25
Purpose: Centralized visual assets for the sales workflow, customer journey, and funnel.
How to use (1-2 lines): Open the SVG/HTML files locally; use this directory to maintain and update visual materials.

- Contents:
  - `sales_workflow.svg`: Complete sales workflow diagram with roles and decision points
  - `customer_journey_map.html`: Interactive customer journey with touchpoints and emotions
  - `sales_funnel.html`: Dynamic funnel visualization by profile with conversion assumptions
  - `branding.md`: Branding guidelines (colors, typography, spacing) for consistency
  - `infographics_overview.md`: Documentation and links for visual assets
  - `version.json`: Simple versioning metadata
