System Overview (5 minutes)
- Objective: Create repeatable pipeline and predictable revenue in 90 days.
- Team: SDRs drive top-of-funnel; AEs run discovery→demo→proposal→close; CSM handles handoff→renewal.
- Process: Lead capture → qualification → outbound cadence → discovery → demo → proposal/negotiation → close → handoff → renewal/expansion.
- KPIs: Leads, SQLs, Win rate, ACV, Pipeline coverage (3x), Sales cycle days, Demo-to-close, Touches to meeting.
- Cadence: 6 touches over 12 days (email, LinkedIn, call); scripts in `/8_TEMPLATES/`.
- Tech: MV (Pipedrive+Gmail+Calendly+Zoom); Recommended (HubSpot+Outreach+Gong).
- Dashboards: Weekly pipeline, activity, conversion; see `/10_KPIS_DASHBOARDS/kpi_definitions.csv`.
- First actions: Load templates; start the outbound cadence; schedule 3 demos; log everything in CRM.
