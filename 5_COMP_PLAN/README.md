Title: Compensation Plan
Owner role: Head of Sales
Last updated: 2025-12-25
Purpose: Define MV and recommended comp plans by role and profile.
How to use (1-2 lines): Select profile; apply MV or recommended; align quotas with KPIs.

Minimum Viable (MV)
- SDR: Base $45k + Variable $15k; quota: 12 SQLs/month.
- AE: Base $80k + Variable $60k; quota: $600k annual bookings.
- Manager: Base $110k + Variable $40k; team quota: 3x pipeline coverage.

Recommended
- SDR: Base $55k + Variable $25k; accelerators on SQLs > 15/month.
- AE: Base $100k + Variable $80k; tiered accelerators beyond 110% of quota.
- Manager: Base $130k + Variable $60k; team attainment accelerator; quality bonus (NPS).
