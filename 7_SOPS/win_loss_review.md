Title: Win/Loss Review SOP
Owner role: Sales Manager
Last updated: 2025-12-25
Purpose: Capture learnings from wins and losses to improve the system.
How to use (1-2 lines): Run monthly; analyze patterns; produce 3 prioritized actions.

1. Collect data — **RevOps** (60m): export wins/losses; annotate reasons.
2. Facilitate review — **Sales Manager** (60m): cross-functional session.
3. Identify patterns — **Sales Manager** (30m): ICP fit, messaging, pricing, competition.
4. Actions & owners — **Sales Manager** (30m): assign owners; due dates.
5. Publish summary — **RevOps** (30m): add to `/16_CONTINUOUS_IMPROVEMENT/`.
