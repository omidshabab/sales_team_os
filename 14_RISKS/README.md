Title: Risks & Mitigations
Owner role: CEO
Last updated: 2025-12-25
Purpose: Identify key risks and mitigation strategies for the sales system.
How to use (1-2 lines): Review quarterly; assign owners for mitigations.

- Risk: Pipeline concentration — Mitigation: diversify sources.
- Risk: Tool sprawl — Mitigation: consolidate; enforce usage.
- Risk: Pricing misalignment — Mitigation: review discount matrix quarterly.
